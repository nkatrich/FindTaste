## Project Description:

The movie explorer "FindTaste". The Application Programming Interface used is ["The Movie DataBase"(TMDB)](https://www.themoviedb.org/)

## Technologies used in the project:

* HTML
* CSS
* JS

*The project uses normalize.css and the Roboto font*

## Coding methods in the project:

### HTML

The HTML code is structured. Semantic tags such as "section," "article" and "main" are used. Classes are primarily used to access tags, and the classes are clearly named. The code at the end of the HTML code is primarily for decoration. Meta tags are used in the head section to improve the site's ranking.

### CSS

The CSS uses modern coding methods, as well as the code itself. The Mobile-First coding method is used, with media query code at the end of the CSS file. Classes are used to apply additional styles that will be called by interactive JavaScript. Skeleton loader was created in CSS, font.css, normalize.css and etc were splitted in files.

### JS

The JavaScript files are divided into different tasks, such as server requests, API data rendering, main.js, config.js for better code understanding. Most variables were created as local scope, Global scope is present, but not a lot. Debounce for input is present. JS modules using such as: import/export, async code uses async/await. There is using a couple of API requests. New movies are loaded when scroll down, thanks IntersectionObserver, that mean scroll is infinite until 76 page. 

## How to run the project:

In my GitHub repository of FindTaste, there is will be description of repository, and inside of description, there is will be link of web-site. If you want open with files, just: In the FindTaste project folder, at the root of the FindTaste folder, there will be an index.html file. Clicking on it will open it. It can be used in any browser. Alternatively, if you have the live server extension, open the FindTaste older in the VC Code editor, open the index.html file, and right-click in the editor. A window will appear with the "Open with Live Server" button.
